安装Centos7

Centos介绍#

Linux系统最重要的就是内核,有了内核之后可以根据自己的需求开发系统。

这种系统被称为Linux发行版,Linux发行版目前常见的有：Debian、Ubuntu、Kali、Centos、Redhat、Fedora...

某些系统是由一个母公司开发的，所以我们称为XXX系列。

  • Ubuntu系列：Ubuntu、Debian、Kali

  • Radhat系列：Centos、Redhat、Fedora

Centos系统是免费的, 而且用于生产环境的软件有很多,所以越来越多的企业喜欢上了Centos系统。

目前主流的Centos版本是7, 最新版本是Centos8。

安装Centos7是运维的基操, 这里就不写了。主要是讲述安装完Centos7的操作

配置静态网络#

原先配置文件/etc/sysconfig/network-scripts/ifcfg-eno16777736

TYPE=Ethernet
BOOTPROTO=dhcp
DEFROUTE=yes
PEERDNS=yes
PEERROUTES=yes
IPV4_FAILURE_FATAL=no
IPV6INIT=yes
IPV6_AUTOCONF=yes
IPV6_DEFROUTE=yes
IPV6_PEERDNS=yes
IPV6_PEERROUTES=yes
IPV6_FAILURE_FATAL=no
NAME=eno16777736
UUID=706ac8a2-1021-412d-8dc7-ff376bfe3e9d
DEVICE=eno16777736
ONBOOT=no

修改主机名#

hostname lofty #临时修改
hostnamectl set-hostname lofty #永久修改

DNS解析#

本地DNS解析文件/etc/hosts

/etc/hosts
172.27.0.5 localhost.localdomain VM-0-5-ubuntu
172.27.0.5 localhost

本地DNS服务器/etc/reslov.conf

在ifcfg网卡文件已经可以指定DNS服务器了, 我们也可以在/etc/reslov.conf指定。但是优先级不清楚。

/etc/reslov.conf
nameserver 114.114.114.114

用于日常巡检的小工具

wget http://percona.com/get/pt-summary
chmod +x ./pt-summary
mv pt-summary /usr/bin

关闭无用服务#

关闭firewalld和selinux#

#临时关闭
systemctl stop firewalld
#永久关闭,即设置开机的时候不自动启动 重启后生效
systemctl disable firewalld
#临时关闭
setenforce 0
#永久关闭selinux
sed -i '/SELINUX/s/enforcing/disabled/' /etc/selinux/config

安装常用软件#

使用Centos yum阿里源#

Centos7是国外开发的, 所以软件包源是国外的, 需要使用国内源进行加速

1. 备份

mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.backup

2. 下载新的 CentOS-Base.repo 到 /etc/yum.repos.d/ 当然如果您没有wget命令可以替代为curl，如果您是6或者7，把Centos-7.repo改成对应数字即可CentOS-Base6.repo即可

curl -o /etc/yum.repos.d/CentOS-Base.repo https://mirrors.aliyun.com/repo/Centos-7.repo
yum -y install epel-release

安装常用软件#

yum -y install git telnet lrzsz vim net-tools lsof ntpdate

安装自动补齐功能#

centos7安装自动补全软件包,这样docker就可以自动补全,重新登陆shell才能生效

yum -y install bash-completion

ulimit 65535#

cat >> /etc/security/limits.conf <<EOF
* soft nofile 65536
* hard nofile 65536
EOF

关闭swap提高性能#

swapoff -a
#UUID=1b7a4d0b-2a57-4d56-b481-45ff65eec35f swap swap defaults 0 0
echo vm.swappiness=0 >> /etc/sysctl.conf
